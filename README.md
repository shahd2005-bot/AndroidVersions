AndroidVersions
This is a simple Android app that shows a list of Android versions like Donut, Eclair, Froyo and more.
What the app does
•	Shows a scrollable list of Android versions
•	Each item has a logo, name, and version number
•	Tap on any item to see its name and version at the bottom of the screen
•	Even and odd rows have different background colors to make the list easier to read
•	There's a sort button that lets you sort the list alphabetically or by version number
How I built it
•	Used RecyclerView to display the list
•	Used CardView for each item in the list
•	Used ViewBinding instead of findViewById
•	Stored the Android version data in a simple class called AndroidVersion
